# Video Thumbnail Downloader

Program sederhana untuk mengambil **thumbnail video** tanpa mengunduh file video utuh.  
Mendukung masukan URL interaktif maupun flag baris perintah.

## ⚙️ Kebutuhan

- [Go](https://golang.org/dl/) 1.16+
- [FFmpeg](https://ffmpeg.org/download.html) (tersedia di PATH)

## 🔧 Cara Menjalankan

1. Simpan kode sebagai `thumbnail.go`
2. Jalankan tanpa argumen untuk mode tanya jawab:
   ```bash
   go run thumbnail.go
